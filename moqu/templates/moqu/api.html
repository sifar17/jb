Hello there!! Thank you for coming this far.

While creating this collection, Moqu felt 🤔 that there might be some people who would need these quotes to use in their own way, on their own platforms or at some place of their own choice.

So, Moqu called me, the API, over and said "Hey API! Give me a hand to fulfill this purpose. Help me to create a medium where people can find my collection of data (quotes along with sources) in a format (json here) that they can use in their own way".

So, here I am with this API room that accomadates all the endpoints. Now, you can access the whole bunch of Moqu's data by reaching out to this room.

As of now, you do not need an API key, you don't need to login or subscribe, just reach the API Room and access the data.

There are three endpoints
Categories GET
You can find all the categories here, their ids and a list of sources belonging to the respective category. You can access a particular category by putting its id at the very end of this endpoint in the url as /endpoint/<id> e.g. /category/1.
Movies GET
You can find all the sources here, their ids, categories and a list of their quotes. You can access a particular movie by putting its id at the very end of this endpoint in the url as /endpoint/<id> e.g. /movie/1.
Quotes GET
You can find all the quotes here, their ids and name of their sources. You can access a particular quote by putting its id at the very end of this endpoint in the url as /endpoint/<id> e.g. /quote/1.

Requests/day: 100 for a unique IP address.
In order to avoid the overcrowding on such a small platform 😞, a limit has been put on the number of times that you can visit the API room which is 100 per day as of now. However, at any time, if you feel this limit to be increased, please send an email to Jugadubhai and I will be more than happy 😃 to increase the limit.

Currently, I am available for GET requests only and not for POST requests i.e. you can access/read all the data but can not update or delete any. However, I will soon add the POST methods as well like Create, Update, Delete, etc.
